FurMark ROG Edition

Visit https://geeks3d.com/furmark/rog/ for more information on this utility and to
download the latest version


System requirements:

- Windows 7 / 8 /10 (32/64-bit)
- GeForce GTX 400 series graphics card or higher.
- Radeon HD 5000 series graphics card or higher.



Revision history:


Version 0.7.0.1 - 2018.06.29
----------------------------
- added new command line options to launch the

Version 0.7.0.0 - 2018.06.21
----------------------------
+ added a new GPU memory stress test:
  rog-07: FurMark(donut)::GPU-Memtest::OpenGL
  This test can utilize up to 90% of the GPU memory.
+ added GLmem utility to monitor OpenGL memory usage.
  Useful with the new GPU memory test.
* fixed a bug related to current app folder.
  Now FurMark-ROG can be launched from any folder 
  via the command line for example.
